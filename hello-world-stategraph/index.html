Hello World StateGraph

A state management library for Terraform. This project uses terraform-provider-aaamockprovider to test StateGraph.

Install

curl -L https://github.com/stategraph/releases/releases/download/0.1.9/stategraph-macos-arm64.tar.gz > ~/Desktop/stategraph-macos-arm64.tar.gz
tar -zxvf ~/Desktop/stategraph-macos-arm64.tar.gz
sudo mv stategraph /usr/local/bin/

Quick Start

# start stategraph server
docker-compose up -d

# open dashboard
open http://localhost:8080

Terraform Usage

export TF_HTTP_USERNAME="session"
export TF_HTTP_PASSWORD="${api_key}"
export STATEGRAPH_API_BASE="http://localhost:8080"
tofu init
tofu plan
