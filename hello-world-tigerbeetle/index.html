Hello World TigerBeetle

A simple example using TigerBeetle, a distributed financial transactions database.

Install

curl -Lo tigerbeetle.zip https://mac.tigerbeetle.com
unzip tigerbeetle.zip
rm tigerbeetle.zip
sudo mv tigerbeetle /usr/local/bin

tigerbeetle version

Quick Start

# create a save file
tigerbeetle format --cluster=0 --replica=0 --replica-count=1 --development ./0_0.tigerbeetle

# start a replica
tigerbeetle start --addresses=3000 --development ./0_0.tigerbeetle
